# AI Test Generation Research Project

A React-based research project for evaluating AI-generated test cases from multiple language models.

## Project Overview

This project contains test files generated by various AI models to compare their testing capabilities:
- Claude (3 test runs)
- Gemini (3 test runs) 
- GPT-4o (3 test runs)
- Grok (3 test runs)
- Llama (3 test runs)

## Setup

```bash
npm install
```

## Available Scripts

### `npm start`
Runs the React app in development mode at [http://localhost:3000](http://localhost:3000)

### `npm test`
Runs all AI-generated test suites

### `npm run build`
Builds the app for production
